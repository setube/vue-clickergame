# 金币点击放置

# 项目介绍
本游戏基于Vue.js开发，使用了PrimeVue和Pinia作为UI库和状态管理库。

- 项目框架 [Vue.js](https://cn.vuejs.org)
- 项目UI [PrimeVue](https://primevue.org)
- 状态管理库 [Pinia](https://pinia.vuejs.org/zh)

# 本地安装

```
npm install
```

## 编译项目

```
npm run build
```

## 运行编译前的项目

```
npm run dev
```

## 运行编译后的项目

```
npm run preview
```
